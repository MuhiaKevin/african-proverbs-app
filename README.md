# African Proverb

Get random qoutes

### Installation
You can install the necessary dependecies by following steps bellow:

```sh
$ git clone https://github.com/MuhiaKevin/african-proverbs-app.git
$ cd african-proverbs-app
```

Run server:
```sh
$ cd server/
$ npm install && npm start
```
Run client:
```sh
$ cd client/
$ npm install && npm start
```
